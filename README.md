# KUBEXPORT

**Kubexport** export Kubernetes resources in structured  yaml or json files.
   Exported resources are stripped of timebased information.

## Requirements
 - **kubectl**

## Quick start
Install `kubexport`:
```
$:~ pip install kubexport
```
