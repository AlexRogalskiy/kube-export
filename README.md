# KUBE-EXPORT

**Kube-export** export Kubernetes resources in structured  yaml or json files.
   Exported resources are stripped of timebased information.

## Requirements
 - **kubectl**

## Quick start
Install `kube-export`:
```
$:~ pip install kube-export
```
